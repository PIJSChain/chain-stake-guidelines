# 截图文件说明

本目录用于存放用户手册中引用的界面截图。

## 需要准备的截图

| 文件名 | 说明 | 对应界面 |
|--------|------|---------|
| `connect-wallet.png` | 连接钱包按钮 | 页面右上角「连接钱包」按钮 |
| `become-validator.png` | 成为验证者入口 | 首页「成为验证者」按钮 |
| `upload-deposit.png` | 上传质押文件 | 质押页面的文件上传区域 |
| `confirm-tx.png` | 确认交易 | 钱包交易确认弹窗 |
| `node-list.png` | 节点列表 | 「我的节点」页面的验证者列表 |
| `get-challenge.png` | 获取Challenge | 「设置便捷管理」中的Challenge值显示 |
| `upload-claim.png` | 上传认领签名 | 认领节点的签名文件上传界面 |
| `add-deposit.png` | 增加质押 | 「增加存款」页面 |
| `stake-orders.png` | 质押订单列表 | 节点仪表板的订单列表 |
| `redeem.png` | 赎回质押 | 「质押赎回」折叠区域展开后的界面 |

## 截图规范

1. **格式**: PNG 格式，透明背景或白色背景
2. **尺寸**: 宽度建议 800-1200px，保持清晰
3. **内容**:
   - 隐藏或模糊敏感信息（钱包地址、余额等可使用示例数据）
   - 确保关键操作元素（按钮、输入框）清晰可见
   - 可使用红色矩形框标注重点区域
4. **命名**: 使用上表中指定的文件名

## 制作建议

- 使用浏览器开发者工具设置固定视口尺寸
- 可使用截图工具的标注功能添加指引箭头
- 建议在测试环境进行截图，使用测试数据
